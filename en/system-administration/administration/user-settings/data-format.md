# Data format

The user has the possibility to adapt the data format.

| Option | Value |
| - | - |
| Date format | English/German |
| Numeric format | English/German |
| Monetary format | `EUR;€` or `USD;$` or `GBP;£` or `CHF;Sfr.` |
